I am a recent graduate of the University of Notre Dame with a major in Mechanical Engineering. My dream is to engage with an elite technology company to design and innovate products that can revolutionize and enhance lives around me.


This is a simple website I have created as a hobby. Recently, I've begun learning Python Flask, a micro-framework used to quickly and easily create webapps running Python on the backend. As you can probably tell by my abundant use of Bootstrap elements, I don't like frontend. But as a hobby I hope to learn more about the backend and create webapps that operate there. This site, for instance, is built from scratch using the Flask infrastructure, which is the Python-based backend framework for this website. It handles browswer requests, directs the user, and can implement any Python-based function. I hope to add lots of fun features to this site such as a login, a blog, tools, custom Python games, and other web-based applications.
